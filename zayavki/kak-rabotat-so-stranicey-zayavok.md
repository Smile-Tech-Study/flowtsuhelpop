title: Как работать со страницей заявок?
---

Страница заявок для быстрого и удобного доступа к любой заявке в системе. По каждому из столбцов доступен быстрый поиск. Для этого надо ввести значение и нажать Enter. 

![](./kak-rabotat-so-stranicey-zayavok-6.png){width=1313px height=221px}

Например, найдем заявку с номером 99:

![](./kak-rabotat-so-stranicey-zayavok-5.png){width=1331px height=264px}

Изначально содержит на странице 6 столбцов по умолчанию: номер заявки, ФИО, указание, что заявка активная, статус заявки, СНИЛС, Email. 

![](./kak-rabotat-so-stranicey-zayavok.png){width=1327px height=458px}

Если необходимо добавить ещё какие-то столбы для отображения, то можно проставить галочку, а также сохранить набор набор этих столбцов в шаблон, чтобы они открывались автоматически. Сделать это можно следующим образом: например, добавили дополнительный столбец Дата рождения, далее нажимаем на кнопку «Сохранить».

![](./kak-rabotat-so-stranicey-zayavok-2.png){width=1317px height=440px}

Называем шаблон для удобства понимания, сохраняем его. 

![](./kak-rabotat-so-stranicey-zayavok-3.png){width=1077px height=186px}

Теперь при открытии страницы заявок будет открываться по умолчанию сохраненный набор столбцов для быстрого доступа. Сохранить можно несколько таких шаблонов для разных случаев, они будут доступны вверху страницы. 

![](./kak-rabotat-so-stranicey-zayavok-4.png){width=320px height=234px}
